# Aula 2.C — APIs REST

Três labs progressivos: consumir uma API existente, criar a sua própria e conectar ao banco de dados.

---

## Labs

| # | Pasta | O que você faz | Tecnologias |
|---|-------|----------------|-------------|
| 1 | [lab1-consumindo-api](lab1-consumindo-api/) | Consome uma API pública com Python | `requests` |
| 2 | [lab2-criando-api-fastapi](lab2-criando-api-fastapi/) | Cria API com GET, POST e DELETE (dados em memória) | `FastAPI`, `uvicorn` |
| 3 | [lab3-api-com-banco-mysql](lab3-api-com-banco-mysql/) | Conecta a API ao MySQL — dados persistem no banco | `FastAPI`, `MySQL`, `Docker`, `.env` |

---

## Por onde começar

Faça os labs **em ordem** — cada um adiciona uma camada sobre o anterior.

1. Clone ou baixe o repositório
2. Entre na pasta do lab
3. Siga o README do lab

---

## Pré-requisitos gerais

- Python 3.10+
- Docker Desktop (só para o Lab 3)
