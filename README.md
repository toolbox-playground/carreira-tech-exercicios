# Carreira Tech — Labs e Exercícios Práticos

Repositório oficial dos labs do curso **Carreira Tech: Do Zero ao Primeiro Emprego em Tecnologia** pela TBXTech.

Cada pasta contém um lab independente com código de exemplo, instruções e checklist para o aluno executar localmente.

---

## Estrutura

```
carreira-tech-exercicios/
└── modulo2-construindo-aplicacoes/
    ├── aula2c-apis-rest/            # requests, FastAPI, Dockerfile
    ├── aula2d-seguranca-basica/     # Credenciais, .env, .gitignore
    └── aula2e-testes-software/      # pytest, pytest-cov, FastAPI TestClient
```
